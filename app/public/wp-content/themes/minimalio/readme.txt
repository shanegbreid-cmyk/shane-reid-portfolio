Minimalio is a minimalist WordPress theme for personal portfolios, it's a lightweight theme with focus on speed, usability and accessibility.

== Description ==
Minimalio is designed specifically for creators, offering all the essential tools to build minimalist portfolio websites.

It leverages WordPress's built-in features, utilizing the Customizer for theme options and Gutenberg for content creation.

Minimalio aims to provide just the essentials for a fast and reliable portfolio, including features like AJAX loading for posts, advanced typography controls, layout controls, and more.

For additional functionality, the Minimalio Premium Plugin is available for purchase, adding a Portfolio custom post type, extra Gutenberg blocks, and additional Customizer options.

= From within WordPress =
1. Visit "Appearance > Themes > Add New"
2. Download the theme from www.minimalio.org or search for "Minimalio"
3. Install and activate

== Frequently Asked Questions ==

= Is Minimalio Free? =
Yes! Minimalio is a free theme.

= Does Minimalio have a pro version? =
Yes, it does! Minimalio has a premium plugin which extends the available options in the theme.

= Where can I find documentation? =
Minimalio has extensive documentation you can find at (https://minimalio.org/blog-instructions).

= Do you offer support? =
Yes, Minimalio offers purchasable support that you can get on the website (https://minimalio.org/support).

= Where can I find the theme options? =
All of our options can be found in the Customizer in 'Appearance > Customize'.

= Does Minimalio have demos for import =
The premium version which is activated by purchasing and activating the Minimalio premium plugin comes with many different demos that could be imported by couple of clicks. You can find the demos at (https://minimalio.org/demos)
